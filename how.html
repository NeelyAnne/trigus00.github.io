How it works?

Instructions

Input a food item into the api search engine. After submitting your search, it sends out a POST api request to Nutritionix Api with your query and returns the results.

The information about the food includes:

  • Calories
  • Sodium
  • Total Fat
  • Saturated Fat
  • Cholesterol
  • Carbohydrate
  • Fiber
  • Sugar
  • Protien
  • Potassium

Tools and Software used:

  • HTML5
  • CSS3
  • Bootstrap v4.0
  • Nutritionix API
  • Javascript ES6
  • Visual Studio
